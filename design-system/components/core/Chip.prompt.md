**Chip** — pill-shaped filter or category toggle used in the Explore filter row and mobile scrollers.

```jsx
<Chip active>All</Chip>
<Chip>Anime</Chip>
<Chip>Celebrity</Chip>
```

`active` → solid gold with dark ink; otherwise a hairline outline that lifts + goes gold on hover. Keep exactly one active in a single-select filter row.
